# College_finder_Tkinter_based
College Finder is a desktop GUI application built using Python (Tkinter) with a MySQL backend. It is designed to help students explore college options based on their chosen stream and course.

The application provides a complete user workflow:
- Sign up and login system
- Stream selection (Engineering / Medical)
- Course-wise college browsing
- Detailed college information (Tier, Target Exam, NIRF Rank, Required Rank, Course Fee, Average Package, Exam Month, Location)
- Search functionality to find colleges by name

In addition to the user features, the system includes an admin mode that allows full database control:
- Add new college records
- Update existing data
- Delete outdated entries
- Search within tables

The project demonstrates practical concepts such as GUI development with Tkinter, database connectivity using MySQL, structured data handling, and role-based functionality.

Developed as part of a Class 12 Computer Science project.
